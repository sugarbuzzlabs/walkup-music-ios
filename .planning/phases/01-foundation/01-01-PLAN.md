  <files>
    project.yml,
    WalkoutDJ/App/WalkoutDJApp.swift,
    WalkoutDJ/Info.plist
  </files>
  <action>
    1. Install xcodegen if not present: `brew install xcodegen`
    2. Remove the existing Expo/React Native boilerplate files (App.tsx, index.ts, package.json, package-lock.json, tsconfig.json, node_modules/, assets/, app.json, .expo/) — this project is pivoting to native Swift
    3. Create project.yml for XcodeGen with:
       - Target: WalkoutDJ (iOS app)
       - Deployment target: iOS 18.0
       - Swift language version: 6
       - Sources: WalkoutDJ/
       - Info.plist with required keys (UILaunchScreen, bundle ID com.walkoutdj.app)
    4. Create WalkoutDJApp.swift as the @main entry point with a basic WindowGroup
    5. Run `xcodegen generate` to produce WalkoutDJ.xcodeproj
    6. Add WalkoutDJ.xcodeproj to .gitignore (generated artifact — project.yml is the source of truth)
    7. Verify build: `xcodebuild -project WalkoutDJ.xcodeproj -scheme WalkoutDJ -destination 'platform=iOS Simulator,name=iPhone 16' build`
  </action>
  <verify>xcodebuild build succeeds with no errors</verify>
  <done>Clean Xcode build, app launches on simulator showing blank screen, Expo boilerplate removed</done>
</task>
