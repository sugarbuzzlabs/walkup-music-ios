  <name>Task 1: Create AudioManager with AVFoundation playback</name>
  <files>WalkoutDJ/Services/AudioManager.swift</files>
  <action>
    Create an @MainActor ObservableObject AudioManager class using AVAudioPlayer:

    Published state:
    - isPlaying: Bool
    - currentTime: TimeInterval
    - duration: TimeInterval
    - volume: Float (0.0-1.0)

    Methods:
    - loadAndPlay(url: URL, startTime: TimeInterval, autoStopAfter: TimeInterval?, volume: Float)
      → loads AVAudioPlayer from URL, seeks to startTime, plays, starts auto-stop Timer if duration provided
    - pause()
    - resume()
    - stop() → stops playback, invalidates timer, unloads player
    - seek(to time: TimeInterval)
    - setVolume(_ volume: Float)

    Implementation details:
    - Use AVAudioPlayer (not AVPlayer — AVAudioPlayer is simpler for local files and supports seeking)
    - Configure AVAudioSession.sharedInstance() for .playback category on first play
    - Auto-stop: use a Timer.scheduledTimer that calls stop() after the configured duration
    - Update currentTime via a displayLink or Timer firing every 0.1s while playing
    - Conform to AVAudioPlayerDelegate to detect playback completion
    - On stop/completion: invalidate all timers, set isPlaying = false
    - Make the class Sendable-safe: store AVAudioPlayer as a property, access only on MainActor
  </action>
